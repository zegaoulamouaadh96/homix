# HomiX - Installation & Setup

## العربية
### المكونات: ESP32 + حساسات + أقفال ذكية + كاميرات RTSP/ONVIF + سيرفر Node.js + MQTT + Flutter

### خطوات التركيب:
1. تركيب الحساسات والأقفال والكاميرات في المواقع المناسبة
2. توصيل ESP32 بالحساسات ثم ربطه بـ Wi-Fi
3. تشغيل MQTT Broker: `mosquitto -v`
4. تشغيل السيرفر: `node server.js`
5. تثبيت التطبيق + إنشاء حساب + إدخال Home Code
6. تفعيل التحقق البيومتري
7. اختبار: أبواب + كاميرات + حساسات + OTP + طوارئ

### ⚠️ تنبيهات
- فتح الباب يتطلب تحقق بيومتري إلزامي
- لا تشارك كود المنزل أو كلمات المرور
- تأكد من وجود بطارية احتياطية
