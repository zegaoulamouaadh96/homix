# 📱 تشغيل التطبيق على الهاتف

## 🌐 الطريقة 1: عبر الويب (الأسهل - أي هاتف)

### خطوات التشغيل:

#### 1️⃣ على الكمبيوتر:
```
افتح المجلد: c:\Users\ADMIN\Desktop\PFE
اضغط مرتين على: RUN-ON-PHONE.bat
```

سيظهر لك عنوان مثل:
```
http://192.168.100.30:8080
```

#### 2️⃣ على الهاتف:
1. **تأكد أن الهاتف والكمبيوتر على نفس شبكة WiFi**
2. افتح المتصفح (Chrome/Safari)
3. اكتب العنوان: `http://192.168.100.30:8080`
4. ✅ التطبيق سيفتح!

---

## 📲 الطريقة 2: تثبيت APK على Android

### الخطوة 1: بناء APK

على الكمبيوتر، افتح PowerShell وشغل:

```powershell
cd c:\Users\ADMIN\Desktop\PFE\frontend
C:\flutter\bin\flutter.bat build apk --release
```

بعد 5-10 دقائق، الملف سيكون في:
```
c:\Users\ADMIN\Desktop\PFE\frontend\build\app\outputs\flutter-apk\app-release.apk
```

### الخطوة 2: نقل APK للهاتف

#### الطريقة A: عبر USB
1. وصل الهاتف بالكمبيوتر عبر كابل USB
2. انسخ ملف `app-release.apk` للهاتف
3. افتح الملف على الهاتف واضغط "تثبيت"

#### الطريقة B: عبر WhatsApp/Email
1. أرسل ملف `app-release.apk` لنفسك عبر WhatsApp
2. حمّل الملف على الهاتف
3. افتحه واضغط "تثبيت"

### الخطوة 3: تفعيل مصادر غير معروفة

إذا ظهرت رسالة "مصدر غير معروف":
1. اذهب إلى: **الإعدادات → الأمان**
2. فعّل: **السماح بالتثبيت من مصادر غير معروفة**
3. أو فعّل للمتصفح فقط

## 🔥 الطريقة 3: Android Debug عبر USB (للتطوير)

### المتطلبات:
- ✅ كابل USB
- ✅ تفعيل USB Debugging على الهاتف

### الخطوات:

#### 1️⃣ تفعيل وضع المطور على الهاتف:
1. إعدادات → حول الهاتف
2. اضغط على "رقم الإصدار" **7 مرات**
3. سيظهر "أنت الآن مطور!"

#### 2️⃣ تفعيل USB Debugging:
1. إعدادات → خيارات المطور
2. فعّل **USB Debugging**

#### 3️⃣ تثبيت Android SDK:
```powershell
# حمّل Android Studio من:
https://developer.android.com/studio

# بعد التثبيت، افتحه وثبت:
# - Android SDK Platform-Tools
# - Android SDK Build-Tools
```

#### 4️⃣ وصل الهاتف وشغل:
```powershell
cd c:\Users\ADMIN\Desktop\PFE\frontend
C:\flutter\bin\flutter.bat devices
# يجب أن يظهر هاتفك في القائمة

C:\flutter\bin\flutter.bat run
# سيثبت التطبيق تلقائياً!
```

## ❓ حل المشاكل:

### "لا أستطيع فتح الرابط على الهاتف"
- ✅ تأكد أن الهاتف والكمبيوتر على نفس WiFi
- ✅ جرب تعطيل Firewall مؤقتاً:
  ```powershell
  # في PowerShell كمسؤول:
  Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled False
  ```
- ✅ جرب عنوان IP آخر (إذا كان عندك أكثر من واحد)

### "التطبيق بطيء على الويب"
- استخدم Chrome على الهاتف (أسرع من Safari)
- أو بناء APK للأداء الأمثل

### "Build APK فشل"
- ثبت Android SDK من Android Studio أولاً
- شغل: `flutter doctor` للتحقق من المتطلبات
