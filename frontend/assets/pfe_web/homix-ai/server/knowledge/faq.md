# HomiX - FAQ الأسئلة الشائعة

## عام
- **ما هو HomiX؟** نظام أمن منزل ذكي شامل يجمع حساسات وكاميرات وأقفال مع ذكاء اصطناعي. أسسه زقاولة معاذ.
- **هل يحتاج إنترنت؟** النظام يعمل محليًا. الإنترنت للتنبيهات الخارجية فقط.
- **التطبيق؟** Android + iOS عبر Flutter.

## أمان
- **البيانات البيومترية؟** تبقى على الهاتف فقط، لا تُرسل للسيرفر أبداً.
- **فتح الباب بدون إذن؟** مستحيل. تحقق بيومتري إلزامي لكل عملية.
- **لو سُرق الهاتف؟** لا يستطيع اللص فتح الباب بدون بصمتك. يمكن إلغاء صلاحيات الجهاز فوراً.
- **التشفير؟** end-to-end لجميع الاتصالات.

## ميزات
- **كم كاميرا؟** 4 كاميرات AI مع إمكانية التوسع.
- **كم باب؟** حتى 3 أبواب ذكية.
- **كود الضيوف OTP؟** 4 أرقام، مرة واحدة، باب محدد، مدة محددة.
- **عند زلزال؟** فتح أبواب تلقائي للإخلاء + تنبيه لكل العائلة.
- **نظام الاحتواء؟** توجيه المتسلل لغرفة + إغلاق أبواب + إنذار. وفق القوانين.
- **الحساسات؟** حركة، دخان، فيضان، كسر زجاج، زلازل، عبث بالجدران.

## تقني
- **المكونات؟** ESP32 + حساسات + أقفال + كاميرات + سيرفر Node.js + MQTT + Flutter.
- **البروتوكول؟** MQTT للسرعة والكفاءة.

## Demo والشراء
- **طلب Demo؟** اسمك + مدينتك + واتساب/بريد + نوع المنزل + عدد الأبواب/الكاميرات.
- **فيديو Demo؟** نعم، فيديوهات 3D لكل ميزة.
- **الباقات؟** أساسية (2 أبواب + 2 كاميرات) / متقدمة (3 أبواب + 4 كاميرات + كل الحساسات) / VIP مخصصة.
- **ضمان؟** نعم، ضمان شامل + دعم فني مستمر.
