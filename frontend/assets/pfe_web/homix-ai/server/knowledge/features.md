# HomiX - Features (AR/FR/EN) — Complete Reference

## العربية
### نظرة عامة
HomiX نظام أمن منزل ذكي متكامل أسسه **زقاولة معاذ**. يجمع بين الذكاء الاصطناعي والأجهزة المتصلة لتوفير أقصى حماية وتحكم.

### 🚪 التحكم بالأبواب
- فتح/إغلاق عن بعد عبر التطبيق مع تحقق بيومتري إلزامي (بصمة/FaceID/PIN)
- دعم حتى 3 أبواب ذكية مع أقفال إلكترونية
- سجل كامل لكل عملية فتح/إغلاق

### 🪟 التحكم بالنوافذ
- فتح/إغلاق بمحرك كهربائي عبر التطبيق
- إغلاق تلقائي عند تفعيل وضع الأمان

### 🔑 أكواد ضيوف OTP
- كود 4 أرقام لمرة واحدة، مدة محددة، باب محدد
- إلغاء فوري + سجل استخدام

### 📷 كاميرات AI (4 كاميرات)
- بث مباشر HD + تعرف ذكي (مصرح/غير مصرح) + كشف حركة + لقطات + تسجيل + رؤية ليلية
- تنبيه فوري عند اكتشاف شخص غريب

### 🌍 حساس زلازل ذكي
- ذكاء اصطناعي يميز الاهتزازات من الزلازل + بروتوكول طوارئ + فتح أبواب تلقائي للإخلاء

### 🧱 كشف العبث بالجدران
- حساسات اهتزاز متقدمة تكشف الحفر/الطرق + تنبيه فوري

### 🏃 حساسات حركة (داخلية وخارجية) مع anti-pet

### 💨 حساسات: دخان + فيضان + كسر زجاج

### 🔒 نظام الاحتواء
- توجيه المتسلل لغرفة احتواء + إغلاق أبواب + إنذار + تنبيه (وفق القوانين)

### 🚨 وضع الطوارئ: زر واحد يقفل كل شيء + إنذار + إشعار + تسجيل من كل الكاميرات

### 👨‍👩‍👧‍👦 إدارة أفراد: مالك/مدير/مقيم/ضيف بصلاحيات مختلفة

### 📱 تطبيق Flutter: Android + iOS، واجهة حديثة، 3 لغات، إشعارات فورية

### 🔐 أمان: تشفير end-to-end، بيومترية على الهاتف فقط، سجل أحداث غير قابل للتعديل
