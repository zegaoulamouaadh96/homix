# HomiX - Security Policy

## سياسة الأمان

### ما نرفضه تمامًا:
- أي طلب لاختراق أو تجسس على أي نظام
- محاولات تجاوز التحقق البيومتري
- طلب فتح باب بدون إذن صاحب المنزل
- مشاركة أكواد OTP أو كلمات مرور
- أي طلب يهدف للإضرار بالآخرين
- طلب معلومات عن ثغرات النظام

### ما نقدمه بدلاً:
- شرح كيفية تأمين منزلك بشكل أفضل
- نصائح لحماية حسابك
- مساعدة في حل مشاكل الأمان بشكل آمن
- شرح كيف يحمي HomiX منزلك

### قواعد أساسية:
- لا تشارك كلمات المرور أبداً عبر الشات
- لا تشارك كود المنزل (Home Code) مع أحد
- لا تشارك أكواد OTP أو رموز التحقق
- استخدم كلمة مرور قوية ومختلفة
- فعّل التحقق البيومتري لكل العمليات الحساسة

### نصائح حماية:
1. غيّر كلمة المرور كل 3 أشهر
2. لا تعطي صلاحيات "مالك" لأي شخص آخر
3. راجع سجل الأحداث بانتظام
4. حدّث التطبيق والسيرفر دائمًا
5. استخدم أكواد OTP محدودة الوقت للضيوف

- Refus total de toute demande de piratage
- Les données biométriques restent sur l'appareil
- Chiffrement end-to-end de toutes les communications

- Absolute refusal of any hacking requests
- Biometric data stays on device
- End-to-end encryption for all communications
